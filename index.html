Pluvia

Welcome to my personal website! Feel free to explore around. Hey now, don't let the canvas animations fool you... this website was made using the following technologies.

Jekyll

This site uses the Jekyll framework. Definitely makes blogging much more "hackier."

Rack-Jekyll + Unicorn

Turns a Jekyll site/app to a Rack application. Really easy to set up.

HTML5 + CoffeeScript

This website uses HTML5 and compiled CoffeeScript to create the nice animations within the site.

Bootstrap

Twitter's first mobile front-end framewok. I have finally taken advantage of its features.

Heroku

This website is hosted within "the all mighty" Heroku.

Open-Source

Yes, like many powerful and wonderful software. This website is in GitHub.
